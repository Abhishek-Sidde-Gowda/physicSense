📡

Passive WiFi Radar

Uses ambient WiFi signals as a bistatic illuminator. No transmitter — completely passive and legal everywhere.

🔊

Acoustic FMCW

Emits an 18–22 kHz ultrasonic chirp via WebAudio API. 4.3 cm range resolution, 3.4 m max range.

🧠

Neuromotor Screening

Classifies tremor type and outputs a proxy UPDRS score for Parkinson's disease screening — no wearable needed.

🔒

Federated Privacy

DP-SGD differential privacy with (ε,δ)-accounting. Gradients shared via WebRTC — raw data never leaves the device.

🗺️

Building Occupancy

Multi-node Kalman tracker assigns persistent IDs to people moving across a whole building floor plan.

🌐

Browser API Standard

Proposes navigator.physicSense as a new Web API — like navigator.mediaDevices but for RF sensing.
